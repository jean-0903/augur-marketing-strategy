§4 — Posicionamento

Modelo MetaTrader

O Augur é o terminal de inteligência e execução. A Polymarket é a corretora.

Augur faz:

  • Onboarding zero-friction (email + PIX)
  • Cria carteira Polygon automaticamente
  • Sinais inteligentes com IA
  • Academy gamificada (20+ missões)
  • Design Zaun/Piltover (cinematográfico)
  • Tudo em português

Polymarket faz:

  • Liquidez dos mercados
  • Execução das operações
  • Custódia (via Polygon)
  • Market making

O usuário nunca sai do Augur. Não precisa saber que "por trás" é Polymarket.

§15 — Riscos e Mitigações

3 riscos críticos

CRÍTICO

Nome "Augur"

Augur Protocol sendo rebootado (Lituus Foundation). Marca encumbered. Risco de trademark.

Ação: Pesquisa INPI + advogado PI ANTES do lançamento. Provável rebrand necessário.

ALTO

Regulatório (Custódia)

Criação de carteira Polygon = possível VASP (Lei 14.478/2022). PIX→USDC = intermediação.

Ação: Wallet-as-a-service de provedor regulado. Exchange parceira para câmbio.

ALTO

Dependência Polymarket

Se Polymarket mudar API ou bloquear acesso, core product quebra.

Ação: Multi-broker (Fase 2). Cache agressivo. Pipeline de dados independente.
